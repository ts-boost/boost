# `@ts-boost/utilities`

> A collection of utility functions to assist development with TypeScript.

## Get It

```shell
npm install --save @ts-boost/core

yarn add @ts-boost/core
```

## Usage

```typescript
import { noop } '@ts-boost/utilities'

const foo: string = 'foo'
const bar: number = 42
const baz: object = {}

noop(foo, bar, baz)
```
